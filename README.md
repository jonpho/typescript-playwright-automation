# TypeScript Playwright Automation Framework

## Tech Stack

- JavaScript/TypeScript
- Playwright Automation

This is a automation framework created in JavaScript/TypeScript using the Playwright Automation library. The structure of the automation framework is based on a Page Object Model to help organize locators.
